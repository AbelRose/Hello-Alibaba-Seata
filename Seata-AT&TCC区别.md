| AT                                 | TCC                  |            |
| ---------------------------------- | -------------------- | ---------- |
| 全局锁                             | 需要                 | 不需要     |
| 回滚日志                           | 需要                 | 不需要     |
| commit/cancel阶段代码实现          | 不需要               | 需要       |
| 是否需要开发者解决悬挂和空回滚问题 | 不需要               | 需要       |
| 性能                               | 低（需要全局锁导致） | 高（无锁） |

TCC场景: 一般用在不支持ACID事务的关系型数据库 (NoSQL) 和 一切场景 (!支持ACID事务的关系型数据库 && 一切场景)
